NexBank Fusion is a premium, modern digital banking web application built with **React**, **TypeScript**, and **Vite**. It features a stunning glassmorphic UI, robust object-oriented banking logic, and a seamless multi-view navigation flow.

## ‚ú® Features

- **Premium UI/UX**: Modern dark mode with glassmorphism effects and smooth animations using Framer Motion.
- **Account Management**: Support for **Savings** (with minimum balance and interest) and **Current** (with overdraft facility) accounts.
- **Secure Flow**: Professional landing page with account verification (Login) and dedicated member dashboards.
- **Persistence**: Your account data remains safe even after closing the browser, thanks to `localStorage`.
- **OOP Core**: Clean architecture using TypeScript classes to implement core banking rules like inheritance and polymorphism.

### Usage Guide
Create Account: Click "Open New Account" on the landing page. Choose Savings or Current and fill in the details.
Login: Click "Already a Member," enter your account number and name to access your dashboard.
Transactions: Use the dashboard to perform deposits and withdrawals. Savings accounts can also apply monthly interest.
Logout: Ready to leave? Click the logout button to securely return to the landing page.
